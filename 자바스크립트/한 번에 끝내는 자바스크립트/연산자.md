### 비교 연산자
`==` 와 `===` 의 차이
- `===` 은 자료형 까지 확인함
- `==` 은 자료형은 비교하지 않고, 값이 일치하는지 확인
```js
let a = 10;

let b = '10';
  
console.log(a == b);
console.log(a === b);
true
false
```
- `!==` 도 마찬가지로 자료형까지 불일치 하는지 확인
- `==` 은 값이 다른지 확인
```js
console.log(a != b);
console.log(a !== b);

false -> 값은 10 10 같은거 아니냐? false
true -> 자료형까지 다름 ok
```
