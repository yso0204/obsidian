> 비 원시 타입 자료형으로 object라고 하며, 여러 개의 데이터 값을 저장할 수 있는 자료형

객체 생성에는 2가지 방법이 있다.
1. 생성자 함수
2. 리터럴
## 생성자 함수
```js
let obj = new Object();
```

## 리터럴
```js
let obj = {};
```

## 객체 속성
> 객체 중괄호 내부에 들어가는 값으로 key, value로 이루어짐

```js
let book = {
    title: "js book",
    author: "sam",
    category: "js",
    year: undefined,
    color: function () {
        console.log('red')
    },
}
```
- 이렇게 함수가 들어가도 됨

### 속성값 사용
> 점표기법 과 괄호 표기법이 있음

`console.log(book.title)`
`console.log(book['title']`
- 괄호의 경우 객체의 key 값을 특정 함수의 매개변수가 결정하거나 key 값이 자주 변경되는 상태에서 주로 사용
```js
const getValue = (key) =>{

    console.log(book[key]);

}
getValue('author');
```
