
# MAC과 IP 주소는 어떻게 연결될까?

Mac 주소는 데이터 링크 계층에서 사용되고, IP 주소는 네트워크 계층에서 사용된다.
**ARP(Address Resoltuin Protocol)** 는 두 계층을 연결하는 역할을 한다.

## 데이터를 전송하는 흐름 확인
