# DHCP 란?
> Dynamic Host Configuration Protocol
- 네트워크에 연결된 기기에 IP주소, 게이트웨이, DNS 서버 같은 네트워크 설정을 자동으로 할당해주는 프로토콜

## 왜 필요한가?
만약 DHCP가 없다면
- 모든 기기에 수동으로 IP주소, 서브넷 마스크, 게이트웨이, DNS를 입력해야함
- 기기가 늘어날 수록 관리가 복잡해지고, 중복 IP 주소가 생길 수 있다

DHCP는 자동화 + 충동 방지 + 편리성 을 제공
