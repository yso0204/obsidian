# VLAN 이란
> Virtual Local Area Network 로 하나의 물리스위치에서 여러 가지 네트워크를 사용할 수 있도록 하는 기술

보통 하나의 스위치에 연결된 기기들은 모두 같은 네트워크 대역에 속하게 되지만, VLAN을 설정하면 마치 여러 대의 스위치가 있는 것처럼 구역을 나눌 수 있다.

# VLAN의 사용 이유
- 보안성 향상
	- 예를 들어 각 부서 별로 네트워크를 격리할 수 있는데, A 팀과 guest Wi-Fi 사용자 간 같은 스위치를 사용하더라도 서로의 데이터에 접근할 수 없게 차단할 수 있다.
- 브로드캐스트 제어
	- 네트워크에 연결된 기기가 많아지면, 불필요한 신호(브로드캐스트)가 전체로 퍼질 수 있는데, VLAN은 이 범위를 좁혀 네트워크의 효율을 높일 수 있다.

# VLAN 동작 방식

## 물리적 포트 기반(Static VLAN)

말 그대로 물리적으로 구분하는 방식으로 하나의 스위치에서 1,2번을 같은 네트워크로 3,4번을 같은 네트워크로 묶는 방식이다.
1,2번과 3,4번은 서로 같은 네트워크 대역에 있지만 서로 직접 통신할 수는 없다.(외부나 내부의 다른 방식으로 해야함)

## MAC 주소 기반(Mac-based VLAN)

MAC 주소를 기준으로 VLAN을 나누는 방식
사용자가 스위치의 어느 포트를 꽂더라도 MAC이 등록되어 있다면 등록된 VLAN 그룹으로 할당됨
